BRAND FONTS — drop your font files here
========================================

The site is wired to use two fonts:

  • Gloudy          → big display headings
  • Century Modern  → body text, nav, labels

These are commercial fonts (Envato), so they aren't bundled. Add the files
to THIS folder using these exact names and the site picks them up automatically:

  Gloudy.woff2          (or .woff / .otf / .ttf)
  CenturyModern.woff2   (or .woff / .otf / .ttf)

Tips:
- .woff2 is best for the web (smallest, fastest). If you only have .otf/.ttf,
  those work too — just keep the file names above.
- To convert .otf/.ttf to .woff2, use a free tool like
  https://transfonter.org or https://cloudconvert.com/ttf-to-woff2

Until the files are added, the site falls back to close stand-ins
(Cormorant Garamond for display, Inter for body) so nothing looks broken.
